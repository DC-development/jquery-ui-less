jQueryUI LESS for TWBS 3.0.0

Make jQueryUI Widgets Blend in with Bootstrap Themes

This is the demopage for jquery-ui-twbs-less. Derived from lphuberdeau's jquery-ui-less.

Play around here switching between the demo themes or go to the project page on github.

It uses:

  • jquery v1.11.0
  • jquery-ui v1.10.4 (custom build with no theme)
  • bootstrap

Files to generate from:

less/jquery.ui.less,
which includes the variables and the jquery-ui-all file. This is the file to compile back to CSS. less/jquery.ui.variables.less,
containing all extracted variables.
